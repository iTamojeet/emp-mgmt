# 🏢 Employee Management System (EMS)

> A lightweight, self-hosted, privacy-first Employee Management System powered by FastAPI and Google Gemini AI — built for small Indian businesses that are priced out of enterprise HR tools.

---

## 💡 Why We Built This

Most HR tools available today — Zoho People, BambooHR, Darwinbox, GreytHR — are:
- **Cloud-only** (your employee data lives on someone else's server)
- **Expensive** (₹500–2000 per employee per month)
- **Bloated** with features that small teams never use
- **Internet-dependent** for every single operation

We built this EMS to solve exactly that problem.

This system runs **entirely on your own machine or server**. Your data never leaves your infrastructure. The AI layer uses Google Gemini to give managers the kind of HR analytics insights that normally require a dedicated analytics team — at a fraction of a rupee per report.

## ⭐ What Makes This Different (USP)

- **Self-hosted & privacy-first** — runs on your own machine, zero data sent to third parties (except Gemini API calls)
- **Gemini AI insights** — weekly, monthly, quarterly team performance reports generated by AI
- **AI Attrition Risk** — Gemini assesses each employee's attrition risk (LOW / MEDIUM / HIGH) based on attendance, tasks and performance
- **360° Performance Reviews** — both manager reviews and employee self-reviews with structured fields
- **Zero vendor lock-in** — open source, runs on MySQL + Python, swap any component
- **Minimal cost** — just your server electricity + Gemini API (fractions of a rupee per report)

## ⚙️ Setup

### 1. Clone and create virtual environment
```bash
git clone <your-repo>
cd emp-mgmt
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 2. Start MySQL via Docker
```bash
# If not already running
docker run --name mysql -e MYSQL_ROOT_PASSWORD=yourpassword -p 3306:3306 -d mysql:latest

# Connect and run schema
docker exec -it mysql mysql -u root -p
# Then paste contents of schema.sql
```

### 3. Configure environment
```bash
cp .env.example .env
# Open .env and fill in your values
```

### 4. Seed dummy data
```bash
python3 seed.py
```

### 5. Run the app
```bash
python3 -m uvicorn main:app --reload
```

Visit: **http://127.0.0.1:8000**

## 🤖 AI Features

### Team Insights
Managers select a period (weekly / monthly / quarterly) and Gemini generates a structured HR analytics report covering:
- Overall team health summary
- Top performers
- Employees needing support
- Attendance concerns
- Task completion analysis
- Actionable recommendations

### Attrition Risk Assessment
Gemini analyzes each employee's last 30 days of data:
- Attendance (present/absent ratio)
- Task completion rate
- Average performance score

And returns for each employee:
- **Risk Level:** 🟢 LOW / 🟡 MEDIUM / 🔴 HIGH
- **Reason:** one-line explanation
- **Recommendation:** one-line action for the manager

### 360° Performance Reviews
- **Manager → Employee:** score (1–10) + notes + period
- **Employee → Self:** achievements + challenges + workload rating (1–5)
- Both visible to manager for a complete picture
